嗨！ 这是一个预约口罩的小爬虫, 将爬取"青岛政务网"公众号,自动预约

- 注意：别改文件名, 别移动文件, 朋友
- 注意：计算机应安装了 python3 和 pip
- 注意：windows 记事本可能会引起编码问题, 本文件夹下所有文件使用utf-8编码, 尽量勿用记事本编辑文件

## 配置文件说明
info.json 为 JSON 文件, 可以用任何编辑器打开 (不推荐记事本)

## 使用教程
1. 连接网络的情况下, 双击 " install.ps1 ", 这将在windows机器中安装本程序的依赖库
2. 打开 " info.json ", 填写预约人信息
3. 打开 " main.py ",
3. 双击 " run.ps1 ", 以运行脚本
