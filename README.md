# 口罩预约程序说明

嗨！ 这是一个预约口罩的小爬虫, 将爬取"青岛政务网"公众号,自动预约

- 注意：别改文件名, 别移动文件, 朋友
- 注意：计算机应安装了 python3 和 pip
- 注意：windows 记事本可能会引起编码问题, 本文件夹下所有文件使用utf-8编码, 尽量勿用记事本编辑文件

## 1. 配置文件说明

```info.json``` 为 ```JSON``` 文件, 可以用任何编辑器打开 (不推荐记事本)

## 2. 使用教程

1. 打开 ```info.json```, 填写预约人信息、预约开始时间
   ( 若当前时间未到 填写的开始时间, 程序将自动等待, 并在填写的时间点开始预约 )

2. 双击 ```run.bat```, 以运行脚本

## 3. JSON 文件说明

```JSON``` 文件不允许注释, 在此处给出说明。
所有可选的街道信息见 ```docs/street.json```

```json
{ 
  // 程序的开始时间, 保持时间在引号内
  "startTime": "09:30",
  // 所有预约信息, 每个 {} 是一条, 彼此用 , 分隔
  "orders": [
    // 第一条信息
    {
      // ... 省略内容
    },
    // 第二条预约信息
    {
      // ... 省略内容
    } 
    // 最后一条信息后方不要有 ,
  ]
}
```
